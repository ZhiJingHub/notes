# 字体文件下载说明

请下载以下字体文件并放入此目录：

## 鸿蒙字体下载链接

从 GitHub 仓库下载：
https://github.com/IKKI2000/harmonyos-fonts

或从华为官方下载：
https://developer.harmonyos.com/cn/docs/design/des-guides/font-0000001157868583

## 需要的字体文件

### 简体中文 (HarmonyOS_Sans_SC)
- HarmonyOS_Sans_SC_Regular.woff2 (400)
- HarmonyOS_Sans_SC_Medium.woff2 (500)
- HarmonyOS_Sans_SC_Bold.woff2 (700)

### 繁体中文 (HarmonyOS_Sans_TC)
- HarmonyOS_Sans_TC_Regular.woff2 (400)
- HarmonyOS_Sans_TC_Medium.woff2 (500)
- HarmonyOS_Sans_TC_Bold.woff2 (700)

### 英文/拉丁文 (HarmonyOS_Sans)
- HarmonyOS_Sans_Regular.woff2 (400)
- HarmonyOS_Sans_Medium.woff2 (500)
- HarmonyOS_Sans_Bold.woff2 (700)

# 关于

这是一个使用 VitePress 构建的精彩文档项目。

## 特性

- 🌍 多语言支持（英文、简体中文、繁体中文）
- 🔍 本地搜索功能
- 🎨 简洁现代的设计
- 📱 响应式布局
- 🚀 部署在 Cloudflare Workers

## 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- [Cloudflare Workers](https://workers.cloudflare.com/) - 边缘计算平台

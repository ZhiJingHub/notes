# 關於

這是一個使用 VitePress 構建的精彩文件專案。

## 特性

- 🌍 多語言支援（英文、簡體中文、繁體中文）
- 🔍 Pagefind 全文搜尋
- 🎨 簡潔現代的設計
- 📱 響應式佈局
- 🚀 部署在 Cloudflare Workers

## 技術棧

- [VitePress](https://vitepress.dev/) - 靜態站點生成器
- [Cloudflare Workers](https://workers.cloudflare.com/) - 邊緣運算平台
